# Inventory System

This is a web-based inventory management system developed using C# and ASP.NET.

## 🚀 Features
- User and role management
- CRUD operations for inventory parts
- Request and transaction tracking
- Real-time reporting

## 🛠 Technologies
- C#
- ASP.NET MVC
- SQL Server
- Bootstrap

## 📌 About the Project
This project was developed as a graduation project and is designed to manage inventory for technical environments. It allows users to manage parts, track transactions, and control access through role-based permissions.
